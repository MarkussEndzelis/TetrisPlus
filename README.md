# TetrisPlus
TetrisPlus is a Windows playable game made with Python. It is an extended version of classic Tetris.

<img width="680" height="643" alt="image" src="https://github.com/user-attachments/assets/49a397e2-421c-471a-8ebc-9e8806db82cf" />


## Features
- Game modes - 3 different game modes
- Hold piece - hold a piece to use it later
- Ghost piece - see where piece will land
- Combo system - score multiplier
- High scores - shows your best scores
- Special piecies - piecies with special effects
- Sound effects - different sound effects

## How to play
Controls:
  leftKey, rightKey - move pieces left and right
  upKey - rotate piece
  downKey - soft drop
  space - hard drop
  c - hold piece

Choose a gamemode from the menu and play

## How to download
1. Download the .exe file from releases
2. Run it
If Windows SmartScreen blocks it, clikc "More info" and then press "Run anyway"

## Tech stack
- Python
